TASK3=> TEMPERATURE CONVERTER

Objective

The objective of this project is to design and develop a Temperature Converter Web Application that allows users to convert temperature values between Celsius (°C), Fahrenheit (°F), and Kelvin (K) quickly and accurately. The application aims to provide a simple, user-friendly, and responsive interface where users can enter a temperature value, select the input and output temperature units, and obtain the converted result instantly. This project also demonstrates the practical implementation of front-end web development concepts such as HTML for webpage structure, CSS for styling, and JavaScript for performing calculations, validating user input, and dynamically updating the webpage. The application is designed to be visually attractive using a modern Glassmorphism design and animated background, enhancing the overall user experience.

Steps Performed
Planned the application layout by identifying the required components, including a temperature input field, two dropdown menus for selecting the source and target temperature units, a Convert button, and a result display section.
Created the webpage structure using HTML by arranging all interface elements inside a card layout to ensure a clean and organized design.
Designed the user interface using CSS by applying a responsive layout, Glassmorphism effect, rounded corners, shadow effects, and an animated gradient background to improve the visual appearance of the application.
Styled the input fields, dropdown menus, buttons, and result section with proper spacing, typography, hover effects, and responsive dimensions to enhance usability.
Implemented JavaScript to read the temperature entered by the user and retrieve the selected input and output temperature units.
Added input validation to ensure that users enter a valid numerical value before performing the conversion. If an invalid value is entered, an appropriate error message is displayed.
Converted the input temperature into Celsius as an intermediate unit using standard mathematical formulas.
Converted the Celsius value into the selected output unit (Celsius, Fahrenheit, or Kelvin) using the corresponding conversion formula.
Rounded the converted temperature to two decimal places for improved readability and accuracy.
Displayed the final converted temperature dynamically within the result section without refreshing the webpage.
Tested the application with different temperature values and conversion combinations to verify the correctness and reliability of the conversion logic.
Ensured the application provides a smooth, responsive, and user-friendly experience across modern web browsers.

Outcome
The Temperature Converter Web Application was successfully developed and tested. The application accurately converts temperature values between Celsius, Fahrenheit, and Kelvin using standard conversion formulas while providing results instantly without requiring a page refresh. The implemented input validation prevents invalid entries and improves the reliability of the application. The modern Glassmorphism interface, animated background, responsive layout, and interactive controls enhance the overall user experience. Through this project, the practical use of HTML, CSS, and JavaScript was demonstrated effectively, including webpage structuring, responsive styling, DOM manipulation, event handling, mathematical computations, conditional logic, and dynamic content updates.
